# Chrome Web Store Test Instructions

Last updated: `2026-04-20`

适用项目：`极简翻译`

## Extension Summary

极简翻译是一个网页划词翻译与朗读扩展。

用户在网页中选中单词或句子后，扩展会在当前页面内直接插入译文，并提供朗读、英文单词词典和设置能力。图片右键 OCR 翻译和多翻译源切换属于同一阅读翻译场景下的辅助能力。

## Reviewer Path

1. 安装扩展后，点击工具栏图标可打开 `popup`。
2. 打开任意普通网页，例如新闻页、博客页或文档页。
3. 用鼠标选中一段英文单词或句子。
4. 页面中会在对应正文位置下直接插入译文结果。
5. 如果选中的是英文单词，可继续打开词典详情查看音标、释义和例句。
6. 在译文卡片中可触发 TTS 播放原文。
7. 打开扩展 `Options` 页面，可查看和保存翻译源、语音参数、白名单/黑名单等设置。
8. 在任意网页图片上点击右键，选择 `翻译图片文字`，可触发图片 OCR 翻译。

## Default Experience

- 默认情况下无需注册账号
- 默认启用至少一个免登录翻译路径
- 用户设置保存在浏览器本地 `chrome.storage`
- 扩展不会要求用户登录后才能体验基础翻译

## Permissions Explanation

- `storage`
  - 用于保存翻译设置、TTS 设置、网站规则和用户输入的第三方接口配置
- `contextMenus`
  - 用于提供网页图片右键翻译入口
- `scripting`
  - 用于在需要时注入或协调页面侧扩展行为
- `host_permissions: <all_urls>`
  - 用于在用户浏览的网页上提供划词翻译、页面内插入译文、图片 OCR 右键翻译，以及访问用户启用的翻译/词典服务

## Network Use

当用户主动使用翻译、词典、OCR 或语法提示功能时，选中的文本或必要请求参数会发送给对应的第三方服务提供商。

默认体验不依赖账号系统。
