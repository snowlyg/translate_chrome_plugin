# 发布检查清单

Last updated: `2026-04-20`

## 1. 上架前资料

- [ ] `manifest.json` 版本号已更新
- [ ] 商店名称、简短描述、详细描述与当前功能一致
- [ ] 隐私政策已发布到可公开访问的固定 URL
- [ ] 支持页已准备好邮箱和 GitHub Issues 入口
- [ ] 已准备 128 图标、截图和宣传图
- [ ] 审核测试说明已整理完毕

## 2. 功能手工验收

- [ ] 普通网页划词可显示页内译文
- [ ] 英文单词可打开词典详情
- [ ] TTS 可正常播放
- [ ] 图片右键 OCR 可触发
- [ ] 设置页保存后，重开浏览器仍生效
- [ ] 至少一个默认翻译路径在当前网络环境可用

## 3. 合规核对

- [ ] 产品定位统一为 `网页划词翻译与朗读`
- [ ] 权限说明与实际行为一致
- [ ] 不宣称账号、同步、订阅等当前未实现能力
- [ ] 不承诺无法保证的稳定性或准确率
- [ ] 第三方服务使用已在隐私政策中说明

## 4. 打包与提交

- [ ] 发布包只包含运行所需文件
- [ ] 未包含本地临时文件、设计草稿和无关素材
- [ ] 已选择 `Delayed publish` 或等效延迟发布选项
- [ ] 已填写测试说明和支持信息

## 5. 发布后两周

- [ ] 每周查看安装量、转化率、评分和评论关键词
- [ ] 汇总重复问题并按 P0/P1/P2 排序
- [ ] 优先处理翻译失败、兼容性和配置丢失问题
- [ ] 至少发 1 个问题修复版本
