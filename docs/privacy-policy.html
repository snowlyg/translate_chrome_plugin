Privacy Policy

Project: Minimal Translation / Last updated: 2026-04-20

Minimal Translation respects user privacy.

This extension is designed to translate selected text on webpages and provide related dictionary and text-to-speech features. The extension does not require account registration and does not intentionally collect personal identity information.

Information We Process

  • Selected text from the webpage, when the user actively chooses text for translation
  • Image URLs or image-related request data when the user actively triggers webpage image OCR translation
  • User settings stored locally in the browser, such as language preferences, enabled translation providers, TTS settings, site rules, and API configuration entered by the user
  • Requests sent to third-party translation, dictionary, OCR, or grammar services when the user uses those features

How Information Is Used

  • Translate selected text
  • Query dictionary information for supported words
  • Recognize and translate text in webpage images when the user requests it
  • Provide grammar hints for supported English sentences
  • Play text-to-speech audio
  • Save user preferences and extension settings

Data Storage

The extension uses Chrome storage to save user settings. These settings may include source and target language preferences, enabled translation providers, TTS voice mode and playback settings, site whitelist and blacklist rules, commercial translation endpoint configuration provided by the user, and other extension preferences required for functionality.

The extension does not maintain its own remote user database.

Third-Party Services

Depending on the user configuration, selected text may be sent to third-party services such as primary free translation endpoints, secondary public translation endpoints, commercial translation endpoints, custom public translation endpoints, dictionary APIs, public OCR services, and public grammar services.

These services are operated by their respective providers and may have their own privacy policies and terms.

Data Sharing

Minimal Translation does not sell user data.

The extension only shares the selected text and required request metadata with the provider needed to fulfill the requested feature.

Minimal Translation does not use the processed data for advertising personalization and does not operate its own remote user account system.

Security

We aim to minimize data usage and only process what is needed for translation and related features. However, no online service can guarantee absolute security.

Children

This extension is not specifically directed to children under 13.
