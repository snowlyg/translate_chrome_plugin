# Privacy Policy

Last updated: `2026-04-04`

Minimal Translation respects user privacy.

This extension is designed to translate selected text on webpages and provide related dictionary and text-to-speech features. The extension does not require account registration and does not intentionally collect personal identity information.

## Information We Process

Minimal Translation may process the following data in order to provide its features:

- Selected text from the webpage, when the user actively chooses text for translation
- User settings stored locally in the browser, such as language preferences, enabled translation providers, TTS settings, and API configuration entered by the user
- Requests sent to third-party translation or dictionary services when the user uses those features

## How Information Is Used

The extension uses this information only to:

- Translate selected text
- Query dictionary information for supported words
- Play text-to-speech audio
- Save user preferences and extension settings

## Data Storage

The extension uses Chrome storage to save user settings. These settings may include:

- Source and target language preferences
- Enabled translation providers
- TTS voice mode and playback settings
- Commercial translation endpoint configuration provided by the user
- Other extension preferences required for functionality

The extension does not maintain its own remote user database.

## Third-Party Services

Depending on the user configuration, selected text may be sent to third-party services such as:

- Primary free translation endpoint
- Secondary public translation endpoint
- Commercial translation endpoint
- Custom public translation endpoint
- Dictionary API

These services are operated by their respective providers and may have their own privacy policies and terms.

## Data Sharing

Minimal Translation does not sell user data.

The extension only shares the selected text and required request metadata with the translation or dictionary provider needed to fulfill the requested feature.

## Security

We aim to minimize data usage and only process what is needed for translation and related features. However, no online service can guarantee absolute security.

## Children

This extension is not specifically directed to children under 13.
