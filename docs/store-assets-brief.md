# Chrome Web Store 素材拍摄与文案脚本

Last updated: `2026-04-20`

适用项目：`极简翻译`

## 素材目标

商店素材只强调一个核心卖点：`网页内直接翻译，不跳页`。

截图和宣传图不要平均分配卖点，优先展示真实阅读场景下的即时翻译体验。

## 图标

- 使用当前 `assets/icons/icon-128.png` 作为商店图标
- 如果后续更换品牌图标，确保与扩展内工具栏图标一致

## 截图建议

### 截图 1：划词即译

- 页面选择：英文技术文档、博客或资讯页
- 画面重点：选中一段英文句子，正文下方直接插入译文卡片
- 标题建议：`划词后直接在网页内看到译文`
- 辅助文案：`不用复制到翻译网站，阅读不断流`

### 截图 2：单词词典

- 页面选择：包含英文单词解释需求的文章页
- 画面重点：选中英文单词并展开词典详情
- 标题建议：`阅读时顺手查看单词释义与例句`
- 辅助文案：`音标、词性、释义和例句集中展示`

### 截图 3：设置页

- 页面选择：`options.html`
- 画面重点：翻译源、语音参数、网站规则
- 标题建议：`按你的阅读习惯配置翻译与朗读`
- 辅助文案：`翻译源、朗读参数与网站生效范围可调`

## 宣传图建议

### 宣传图主文案

`网页内直接翻译，不跳页`

### 宣传图副文案

`选中单词或句子后，直接在当前页面查看译文、朗读与词典信息`

### 画面建议

- 左侧放网页正文和选区
- 中间突出页内译文卡片
- 右侧用简短标签点出 `词典`、`朗读`、`OCR`
- 避免在宣传图里堆太多设置项或翻译源名称

## 拍摄注意事项

- 尽量使用干净的网页内容，避免版权风险过高的页面截图
- 保持浏览器缩放统一，避免不同截图字号跳变
- 截图里尽量不要出现与产品无关的个人信息、聊天窗口、书签内容
- 如果截图用于中文市场首发，优先中文标题和说明
