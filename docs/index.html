Chrome Extension

极简翻译

一个面向网页阅读场景的划词翻译与朗读扩展。选中单词或句子后，直接在当前页面内查看译文、朗读和词典信息，不需要复制到外部翻译网站。

隐私政策 支持与反馈 GitHub 仓库

核心定位

单一目的：网页划词翻译与朗读。

适合阅读英文技术文档、博客、资讯和跨语言网页内容时，减少复制粘贴和标签页切换。

主要能力

页内即时翻译

选中单词或句子后，译文直接插入到当前网页内容附近。

朗读与词典

支持 TTS 播放，英文单词可查看音标、释义、例句等词典信息。

扩展阅读辅助

支持图片 OCR、多翻译源切换和网站生效规则，用于补强网页阅读体验。
